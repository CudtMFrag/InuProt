# InuProt — VS Code Theme Pack

Two paired themes:

- **InuProt Dark** — black, high-contrast, ink-green keyword/operator palette.
  Forked from the **Human High Contrast** theme in [Human Theme](https://github.com/tom-f-hall/human-theme-vscode)
  by Tom Hall, with its green ramp dimmed for better separation from the orange
  control-flow / string color.
- **InuProt Light** — soft green-tinted background chrome.
  Forked from VS Code's built-in **2026 Light** theme (Microsoft, MIT), with the
  editor surface retinted to `#edf3eb` and chrome to `#e6eee4`; syntax colors
  are inherited unchanged from the upstream `light_modern → light_plus → light_vs`
  chain.

Built via `@vscode/vsce`.
